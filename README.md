Persian:
پیاده سازی روش های محاسبه دترمینان: 1- روش بازگشتی یا بسط لاپلاس 2- روش حذفی گاوس-جردن 3- روش مقاله امید رضایی فر و در انتها پاده سازی روش مقاله امید رضایی فر برای رمز نگاری در hill cipher
English:
Implementation of determinant computation methods:

1- Recursive method (Laplace expansion)

2- Gaussian–Jordan elimination method

3- The method proposed in Omid Rezaei Far’s paper(A new approach for finding the determinant of matrices)

Finally, implementation of Omid Rezaei Far’s paper method for encryption in the Hill cipher.
